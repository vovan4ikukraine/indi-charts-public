# Screenshots

Place your application screenshots in this directory.

## Recommended Screenshots

1. **home-screen.png** - Main screen with indicator chart
2. **chart-view.png** - Detailed chart view with indicators
3. **alerts-screen.png** - Alerts management screen
4. **watchlist-screen.png** - Watchlist with multiple instruments
5. **alerts.png** - Alerts
6. **widget.png** - watchlist widget
8. **markets-screen.png** - Markets browser screen
9. **settings-screen.png** - Settings screen

## Image Guidelines

- **Format**: PNG or JPG
- **Size**: Recommended 400x800px (portrait) or 800x400px (landscape) for mobile screenshots
- **Naming**: Use lowercase with hyphens (e.g., `home-screen.png`)
- **Quality**: Use high-quality images to showcase your app
